User request hits AgentCore Runtime (Fargate). Bedrock Claude 3.5 Sonnet handles reasoning. Strands SDK orchestrates the agent loop.
2 MCP Tool Call
Agent decides to query database. MCPClient calls Teradata MCP Server via JSON-RPC over stdio.

Response
Results flow back: Teradata → MCP → Agent → User. Full round-trip.
